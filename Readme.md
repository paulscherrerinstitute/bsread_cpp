# Overview
CRLOGIC provides a fast IOC based continuous scan functionality. It supersedes the G/OTFSCAN project.

# Development

* Compilation

```
make
```

* Installation

```
make install
```

# Debugging
Start medm panel:
```
medm -x -macro "P=MTEST-HW3-CRL" /work/sls/config/medm/G_CRLOGIC_expert.adl
```
